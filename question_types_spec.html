公共字段（所有题型共享）
COMMON FIELDS · SHARED BY ALL TYPES
📐字段清单
字段名类型必填说明
id string ✓ 唯一 ID，建议格式：ruxinglun_c{品号}_v{颂号}_{序号}
type enum ✓ 题型 code：single/fill/match/sort/multi/open/image/listen/flip/flow/guided/scenario
typeLabel string ✓ 前端显示的题型名称（如"单选题"）
chapter int ✓ 入行论品号 1-10
verseRange string 对应颂词范围，如"11-18"
lessonNum int 班级课程第几天/课，用于按班级进度推送
difficulty int ✓ 难度等级 1-3（入门/进阶/高阶）
tags string[] 主题标签，如 ["菩提心","七支供养","忏悔"]
correctText string ✓ 答对后的讲解，50-150 字
wrongText string ✓ 答错后的正解和引导，50-150 字
source string ✓ 出处，精确到品名和颂号
createdAt datetime ✓ 创建时间
reviewed bool ✓ 是否已经过法师/编辑审核
📌命名规范
类型 code全小写英文，单个单词优先
字段名驼峰命名（camelCase）
ID 格式下划线分隔，便于数据库索引
时间字段ISO 8601 格式 (2025-01-20T10:30:00Z)
1
单选题
SINGLE CHOICE · type: "single"
v1.0
🎯适用场景
作者朝代 经名识别 概念辨析 基本事实 义理选择
📝专属字段
字段名类型必填说明
question string ✓ 题目文本
options Option[] ✓ 选项数组，固定 4 个
options[].text string ✓ 选项文字
options[].correct bool ✓ 是否正确，有且仅有 1 个 true
💎完整 JSON 示例
{ "id": "ruxinglun_c3_v1_001", "type": "single", "typeLabel": "单选题", "chapter": 3, "verseRange": "1-8", "lessonNum": 3, "difficulty": 1, "tags": ["七支供养", "受持菩提心"], "question": "《入菩萨行论》第三品主要修持什么？", "options": [ { "text": "七支供养", "correct": true }, { "text": "十不善业", "correct": false }, { "text": "四圣谛", "correct": false }, { "text": "十二因缘", "correct": false } ], "correctText": "第三品核心为修持七支供养——礼敬、供养、忏悔、随喜、请转法轮、请佛住世、回向。", "wrongText": "正解：七支供养。第三品正是通过七支供养圆满积资净障。", "source": "出处：《入菩萨行论》第三品", "createdAt": "2025-01-20T10:30:00Z", "reviewed": true }
出题建议：四个选项要合理分布难度，不能一眼看出正确答案。干扰项最好用"看似正确实则错误"的常见误解（如将世俗菩提心和胜义菩提心混淆）。
2
填空题
FILL IN BLANK · type: "fill"
v1.0
🎯适用场景
颂词背诵 名号记忆 经咒关键字 对仗词汇
📝专属字段
字段名类型必填说明
question string ✓ 题目说明，如"补全颂词"
verseLines string[] ✓ 颂词数组，用 "___" 标记空位
verseSource string ✓ 颂词出处小字展示
correctWord string ✓ 正确答案
options string[] ✓ 候选词数组（含正确答案在内共 4 个）
💎完整 JSON 示例
{ "id": "ruxinglun_c1_v20_fill_001", "type": "fill", "typeLabel": "填空题", "chapter": 1, "verseRange": "20-22", "difficulty": 1, "tags": ["颂词", "菩提心"], "question": "补全颂词", "verseLines": [ "众生欲除苦", "反行痛苦因", "愿求安乐故", "愚昧毁___" ], "verseSource": "《入菩萨行论》第一品", "correctWord": "自乐", "options": ["自乐", "他乐", "安乐", "善乐"], "correctText": "众生因愚昧而自毁幸福——想要快乐却做招苦的事。", "wrongText": "正解：自乐。寂天菩萨在此揭示凡夫颠倒的行为。", "source": "出处：《入菩萨行论》第一品·颂 28", "createdAt": "2025-01-20T10:30:00Z", "reviewed": true }
注意：v1.0 只支持一个空。如果颂词有多个关键词需要填，建议拆成多道填空题。多空填空留到 v1.1 再做。
3
连线配对
MATCHING · type: "match"
v1.0
🎯适用场景
名相释义 波罗蜜对治 三宝内涵 法器象征 十不善业果报
📝专属字段
字段名类型必填说明
question string ✓ 题目说明
left Item[] ✓ 左列项目，通常 4 个
left[].id string ✓ 左项 ID（a/b/c/d）
left[].text string ✓ 左项文本
right Item[] ✓ 右列项目，与左列一一对应
right[].id string ✓ 右项 ID（1/2/3/4）
right[].text string ✓ 右项文本
right[].match string ✓ 指向左项 id，建立配对关系
💎完整 JSON 示例
{ "id": "ruxinglun_c3_match_001", "type": "match", "typeLabel": "名相配对", "chapter": 3, "difficulty": 2, "tags": ["七支供养", "对治烦恼"], "question": "将七支供养与其对治的烦恼配对", "left": [ { "id": "a", "text": "礼敬支" }, { "id": "b", "text": "供养支" }, { "id": "c", "text": "忏悔支" }, { "id": "d", "text": "随喜支" } ], "right": [ { "id": "1", "text": "对治骄慢", "match": "a" }, { "id": "2", "text": "对治悭贪", "match": "b" }, { "id": "3", "text": "净除业障", "match": "c" }, { "id": "4", "text": "对治嫉妒", "match": "d" } ], "correctText": "七支供养对治七种烦恼：礼敬对骄慢，供养对悭贪，忏悔对业障，随喜对嫉妒。", "wrongText": "正解：礼敬-骄慢，供养-悭贪，忏悔-业障，随喜-嫉妒。", "source": "出处：《入菩萨行论》第三品", "createdAt": "2025-01-20T10:30:00Z", "reviewed": true }
UI 注意：左右列数量必须相等。右列 text 在渲染时会被随机打乱顺序，左列保持原序。
4
排序题
SORTING · type: "sort"
v1.0
🎯适用场景
修行次第 品目顺序 十二因缘 三士道 仪轨步骤
📝专属字段
字段名类型必填说明
question string ✓ 题目说明
items Item[] ✓ 待排序项数组，3-6 个
items[].text string ✓ 项目文字
items[].order int ✓ 正确序号（1-N）
💎完整 JSON 示例
{ "id": "ruxinglun_c3_sort_001", "type": "sort", "typeLabel": "排序题", "chapter": 3, "difficulty": 2, "tags": ["七支供养", "次第"], "question": "请按七支供养次第排列", "items": [ { "text": "礼敬支", "order": 1 }, { "text": "供养支", "order": 2 }, { "text": "忏悔支", "order": 3 }, { "text": "随喜支", "order": 4 } ], "correctText": "七支次第：礼敬→供养→忏悔→随喜→请转法轮→请佛住世→回向。不可颠倒。", "wrongText": "正解：礼敬-供养-忏悔-随喜。先净障再积资，后随喜回向。", "source": "出处：《入菩萨行论》第三品·七支次第", "createdAt": "2025-01-20T10:30:00Z", "reviewed": true }
UI 注意：前端渲染时必须把 items 打乱显示。用户点击项目后标记 1、2、3、4。建议 items 数量 4 个为最佳，过多（如 12 因缘）对用户挑战太大，可拆成两道小题。
5
判断多选
MULTIPLE CHOICE · type: "multi"
v1.0
🎯适用场景
列举型思考题 全面掌握测试 属性归类 反例识别
📝专属字段
字段名类型必填说明
question string ✓ 题目，结尾注明"（多选）"
options Option[] ✓ 选项数组，6-8 个（多于单选）
options[].text string ✓ 选项文字
options[].correct bool ✓ 是否正确，可多个 true
scoringMode enum ✓ 评分方式：strict（全对才对）/ partial（按比例给分）
💎完整 JSON 示例
{ "id": "ruxinglun_c1_multi_001", "type": "multi", "typeLabel": "判断多选", "chapter": 1, "difficulty": 2, "tags": ["菩提心", "利益"], "question": "下列哪些属于菩提心的利益？（多选）", "options": [ { "text": "净除无量劫罪业", "correct": true }, { "text": "增长世间财富", "correct": false }, { "text": "成佛的唯一因", "correct": true }, { "text": "凡夫转为佛子", "correct": true }, { "text": "能获得神通", "correct": false }, { "text": "圆满福德资粮", "correct": true } ], "scoringMode": "strict", "correctText": "菩提心四大利益：净罪业、圆福德、成佛之因、转凡为佛子。世间福报和神通非其核心价值。", "wrongText": "正解：净罪业 · 成佛因 · 转凡为圣 · 圆福德。其他属于世间法不列其中。", "source": "出处：《入菩萨行论》第一品·菩提心利益", "createdAt": "2025-01-20T10:30:00Z", "reviewed": true }
评分建议：v1.0 统一用 strict 模式（全选对才算对），简单稳定。v1.1 可引入 partial 模式，按正确选中比例给分。干扰项要有迷惑性，不要太明显。
6
开放作答
OPEN-ENDED · type: "open"
v1.0 🌟
🎯适用场景
课后思考题 义理阐述 "为什么" "如何理解" 综合分析
📝专属字段
字段名类型必填说明
question string ✓ 思考题原文
referenceAnswer string ✓ 法师标准答案全文，给 AI 和用户参考
keyPoints string[] ✓ 答案关键要点列表，用于 AI 评分
minLength int 最少字数建议，默认 50
maxLength int 最多字数建议，默认 500
gradingHint string 给 AI 的额外评分提示（可选）
strictMode bool 是否严格评分，默认 false
💎完整 JSON 示例
{ "id": "ruxinglun_c1_open_001", "type": "open", "typeLabel": "思考题", "chapter": 1, "lessonNum": 1, "difficulty": 2, "tags": ["菩提心", "大乘根本"], "question": "为什么说菩提心是大乘佛法的根本？", "referenceAnswer": "菩提心是为利益一切众生而发愿成佛的心...（法师完整答案）", "keyPoints": [ "菩提心的定义：为利众生愿成佛", "是成佛之因，无菩提心不能成佛", "能净除无量劫罪业", "能圆满福德资粮", "令凡夫转为佛子，堕入佛种性" ], "minLength": 80, "maxLength": 400, "gradingHint": "重点看是否答到菩提心的利他特质和与小乘的区别", "strictMode": false, "correctText": "菩提心之所以为大乘根本，在于它既是成佛之因，又是净罪积福之源。", "wrongText": "建议重读法师标准答案，重点理解菩提心与六度的关系。", "source": "出处：《入菩萨行论》第一品·思考题", "createdAt": "2025-01-20T10:30:00Z", "reviewed": true }
🤖AI 评分返回结构
// AI 评分后返回，存入答题记录 { "score": 72, "level": "良好", "hitPoints": [ { "point": "菩提心的定义", "hit": "full", "userPhrase": "..." }, { "point": "能圆满福德", "hit": "partial", "userPhrase": "..." } ], "missedPoints": ["凡夫转为佛子"], "encouragement": "您对菩提心的基本利益已掌握 ✨", "suggestion": "可以补充：初发菩提心即转凡夫身为佛子。", "misunderstanding": "" }
核心价值：这是 v1.0 最大的差异化题型。所有课后思考题都应该用此类型录入。详细的评分 Prompt 设计见独立文档。

情境判断
SCENARIO JUDGMENT · type: "scenario"
v2.0
🎯适用场景
戒律应用 日常行持 修行对治 实务智慧
📝专属字段
{ // 公共字段同上 "type": "scenario", "typeLabel": "情境判断", "scenario": "有人无缘无故在公共场合辱骂你，你内心升起愤怒。作为菩萨道修行者，应如何以六度对治？", "scenarioImage": "optional-illustration.jpg", "options": [ { "text": "以安忍波罗蜜观照，这是消业的机会", "correct": true, "reason": "正确：嗔心生起时正应修安忍" }, { "text": "当场反驳以正视听", "correct": false, "reason": "增长嗔心，失菩提心" }, { "text": "默默走开不予理会", "correct": false, "reason": "回避但未真正对治" }, { "text": "以慈悲心观对方是业障所缚", "correct": true, "reason": "正确：以慈悲观转嗔为悲" } ] }

📚 文档维护规则

• 所有新增题型必须先更新本文档再实现代码

• JSON Schema 字段变更要同步更新题库生成脚本、题库编辑器、前端渲染

• 公共字段修改需评估对所有题型的影响

• 题型 code 一旦确定不可更改（避免数据迁移）

• 每个题型上线前必须有至少 20 道样例题作为测试数据
